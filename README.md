## Summary
Library of different Roleplay Tabletop Game 'objects'. Users can access 'public' objects, made by other users, and create their own, public or private.

### Functions :
  - Admin : 
    - create new objects
    - see users
  - Users :
    - rent objects
    optional : create a character ?
